# AgentRun 是什么

AgentRun 是一站式的 AI Agent 基础设施平台，为企业级 Agent 提供从开发、部署到运维的全生命周期支持。

它以函数计算（FC）为底座，继承了极致弹性、按量付费、零运维的核心优势。AgentRun 针对 Agent 场景提供高性能 Sandbox 与企业级隔离、模型统一代理与治理、全链路可观测与成本管控、工具与 MCP 统一管理、以及完善的安全治理能力。平台深度集成主流开源生态，支持一键部署托管，并可模块化按需使用，与已有系统灵活对接。通过深度集成云原生服务，让开发者专注于业务逻辑，扫清 AI Agent 规模化落地中的基础设施障碍。
